# 通用权限管理系统 java950

## 项目概述

本项目是一款基于Vue3+SpringBoot的前后端分离的通用权限管理系统。系统提供了一套完整的企业级动态权限解决方案，支持菜单、路由、按钮的全控制，同时集成专业的可视化图表，以TypeScript实战教学形式，完美搭配Vue3技术栈。

## 技术栈

### 前端

- Vue3.2语法糖
- TypeScript
- Pinia
- 动态路由/菜单
- ECharts
- 权限控制

### 后端

- SpringBoot
- SpringSecurity
- JWT Token认证

## 功能模块

- 企业级动态权限方案
- 专业可视化图表集成
- TypeScript实战教学
- SpringSecurity前后端分离安全架构
- 全流程开发教学（含业务逻辑抽离技巧）

## 系统角色

本项目适用于以下角色：

- 想提升全栈能力的中高级开发者
- 需要计算机设计的计算机专业同学
- 准备面试项目经验的求职者

## 运行环境

- 前端：建议使用现代浏览器，如Chrome、Firefox等。
- 后端：Java 8或更高版本，SpringBoot框架。

## 部署步骤

1. 准备开发环境（Node.js、Java、IDE等）
2. 克隆或下载项目源码
3. 配置数据库和后端环境
4. 运行前端开发服务器
5. 运行后端服务
6. 访问系统进行测试
